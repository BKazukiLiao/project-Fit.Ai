# project-fit.ai
Fitness Application Built off of CHATGPT Api

# what does it do?
- creates workout plan catered to each specific persons needs
  Factors may include: 
    - Equiptment Availability
    - Injuries  
    - Bodypart focus
    - Goals
- creates a meal plan catered to an individuals goals
  Factors may include:
   - Dietary Restrictions
   - Preferences
   - Budget
